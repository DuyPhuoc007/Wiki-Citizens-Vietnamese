# Wiki-Citizens-Vietnamese
Wiki Citizens tiếng Việt

Tải Citizens [tại đây](https://www.spigotmc.org/resources/citizens.13811/).
Đặt file Citizens 2.0 (.jar) vào thư mục /plugins của bạn.
Tải các plugin Trait mà bạn muốn và cũng đặt chúng vào thư mục /plugins.
Reload hoặc khởi động lại server của bạn. Citizens 2 sẽ tạo thư mục plugins/Citizens và file config.yml.
Cấu hình Citizens theo ý bạn bằng file config.yml. Xem trang Configuration để được trợ giúp thêm.
Reload hoặc khởi động lại server.
Bắt đầu tạo và chỉnh sửa NPC!
Đừng quên thiết lập Permissions để người chơi có thể tương tác với NPC.
Lưu ý rằng người chơi cần quyền citizens.npc.talk để có thể nghe NPC nói chuyện!
